# LocalPictureSaver iOS App

A SwiftUI iOS app to select photos and upload them to your own server.

## Features
- Select multiple photos from your library
- Upload to your own FastAPI server
- Configure server address and API token

## Usage
- Open the app, select photos, and tap "Upload Selected"
- Set your server address and token in Settings

## Requirements
- iOS 16+
- Swift 5.9+

## Note
- This project is designed for manual Xcode import. Some features (like SwiftUI previews) may not work outside Xcode.
